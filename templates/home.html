Weather Data Api

  1. URL format for one station and a specific date.: http://127.0.0.1:5000/api/v1/station/date

    URL example: http://127.0.0.1:5000/api/v1/10/1988-10-25

  2. URL format for one station and every date: http://127.0.0.1:5000/api/v1/station

    URL example: http://127.0.0.1:5000/api/v1/10

  3. URL format for one station for a year: http://127.0.0.1:5000/api/v1/yearly/station/date

    URL example: http://127.0.0.1:5000/api/v1/yearly/10/1988
